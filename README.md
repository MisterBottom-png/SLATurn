# Kuluplanner Turnover & SLA Dashboard

This repository now contains a static, GitHub Pages-friendly React + TypeScript dashboard for analyzing turnover and SLA performance from Excel workbooks.

## Run locally

```bash
npm install
npm run dev
```

## Build for GitHub Pages

```bash
npm install
npm run build
```

The Vite config is set to `base: './'` so the build output is deployable to GitHub Pages (or any static hosting) without additional rewrites.

## Tests & lint

```bash
npm run test
npm run lint
```
